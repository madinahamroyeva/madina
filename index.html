CSS O'rganamiz

Css xususiyatlari:

  • Rang: color-matn rangini belgilaydi.
    • Orqa fon: background-color-orqa fon rangini belgilaydi.
    • Hajm: widh, height-elementning kengligi va balandligini belgilaydi.
    • Matn: text-align-matnni markazga yoki chap/ong tomonga joylashtirish.
    • Chegaralar:border-elementga chegaralar qo'shish.
    • Yumaloqlik: border-radius-burchaklarni yumaloqlashtirish
    • Ro'yxat: list-style-ro'yxat belgilarini o'zgartirish.
    • Animatsiya: transfrom, transition-harakat va o'zgarishlar qo'shish.
    • Flexbox: display: flex, justify-content, alkign-items- elementlarni moslashuvchan tartibda joylashtirish.
    • Grid:display; grid, grid-template-columns- tarmoq joylashuvi yaratish.
